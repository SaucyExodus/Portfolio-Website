Projects

Task Management with Slack Application

As a team we designed and developed "The Task App," an innovative application aimed at boosting team productivity by leveraging the Slack API.

Key Features and Implementations:

  • Kanban-Style Task Management: Enables users to create, edit, and delete tasks using a Kanban board interface, facilitating efficient task management and visualization.
  • Seamless Firebase Data Storage: Utilizes Firebase for reliable and secure data storage, ensuring that all task-related information is easily accessible and synchronized.
  • Cross-Platform Compatibility: Built with cross-platform compatibility in mind, allowing users to access and manage tasks from various devices seamlessly.
  • Google Calendar Integration: Integrates smoothly with Google Calendar, enabling users to sync tasks and deadlines, ensuring better scheduling and time management.
  • Intuitive User Interface: Designed with a user-friendly interface that makes navigation and task management straightforward and efficient.

Robot Obstacle Avoidance

I developed a sophisticated program specifically designed for the FRDM-KL46z microcontroller board. The project involved creating an autonomous obstacle avoidance system, utilizing the board's extensive functionalities and peripherals.

Key Features and Implementations:

  • Autonomous Obstacle Avoidance: Leveraged the microcontroller's capabilities to develop a system that autonomously detects and avoids obstacles.
  • Motor Control: Implemented motor control functions to enable smooth movement in various directions, ensuring agile navigation.
  • Servo Motor Configuration: Configured and controlled a servo motor using pulse width modulation (PWM) for precise manipulation and directional control.
  • Distance Measurement with Sonar Sensor: Handled interrupts to accurately measure distances using a sonar sensor, enhancing the system's obstacle detection capabilities.
  • User-Initiated Switch: Integrated a user-initiated switch for seamless program activation, providing an intuitive start mechanism for the system.

Computer Networking Socket Project

I developed a comprehensive program to manage a peer-to-peer distributed hash table (DHT) network, ensuring efficient data distribution across all nodes and handling all network conditions.

Key Features and Implementations:

  • DHT Network Management: Oversaw peer registration, DHT setup, maintenance, and teardown to ensure a robust and reliable distributed network.
  • Dynamic Peer Selection and Leader Election: Designed and implemented a DHT system to manage dynamic peer selection, registration, and leader election, enhancing network reliability.
  • Socket-Based Communication: Utilized socket-based communication for peer commands, ensuring a stable and proper connection for data transfer.
  • Advanced Algorithms: Applied advanced algorithms for peer selection, load balancing, and data hashing, optimizing performance in a distributed network environment.
